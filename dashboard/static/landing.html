Core capabilities

Designed for Production, Not Demos

Zero-Dependency Runtime

Run the queue server with no required Redis, RabbitMQ, or external broker.

Retries + Backoff + DLQ

Handle failure with fixed, linear, or exponential retries and inspect dead-letter jobs.

Progress + Heartbeats

Track long-running work and detect stalls before they become incidents.

Real-Time Telemetry

Stream job lifecycle events via WebSocket and scrape metrics via Prometheus.

Embeddable Rust Library

Use RustQueue as infrastructure or embed it directly into Rust services.

Single-Binary Deployment

Ship one binary from local dev to edge devices to production clusters.

Where it fits

Built for Teams Shipping Asynchronous Workloads

SaaS Background Jobs

Emails, webhooks, report generation, billing retries, and media processing.

AI Task Pipelines

Long-running inference and agent work with progress events and retries.

Edge and On-Prem

Minimal dependencies for constrained environments and regulated workloads.
